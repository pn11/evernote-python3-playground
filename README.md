# Evernote Python3 API のおためし

## Usage

!!Caution!! This runs in your Evernote DB, not a sandbox.

```sh
$ cat .env
EVERNOTE_TOKEN="YOUR_TOKEN_HERE"
$ pip3 install requirements.txt
$ python3 EDAMTest.py
```
